# SQL Covid Project

Followed a tutorial by [Alex The Analyst](https://www.youtube.com/watch?v=qfyynHBFOsM) to import raw csv files into a database and performed complex queries to extract meaningful public health information regarding COVID-19. 

## Skills used

Data joins, aggregate functions such as SUM, MAX, and AVG, CTE, windows function such as PARTITION BY, and saving queries as Views for later use in data visualization.
